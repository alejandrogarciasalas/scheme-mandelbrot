# scheme-mandelbrot
My UC Berkeley's CS61a recursive art contest submission

This is a mandelbrot set written in scheme (a dialect of Lisp). The code is meant to be run with a custom interpreter we had to make for CS61A. This interpreter would interface with the Python's turtle graphic module.
